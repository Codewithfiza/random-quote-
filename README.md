# 💬 Random Quote Generator

A beautifully designed random quote generator built with pure HTML, CSS, and vanilla JavaScript — no frameworks, no libraries.

This is **Project 1 of 20** in my JavaScript learning journey.

---

## 🚀 Live Demo

> Coming soon — will be deployed via GitHub Pages

---



## ✨ Features

- 20 curated quotes from world-famous personalities
- Randomly picks a new quote on every button click
- Displays the quote author and category tag
- Copy quote to clipboard with one click
- Share directly to Twitter / X
- Smooth fade animations between quotes
- Fully responsive — works on mobile and desktop

---

## 🛠️ Built With

- HTML5
- CSS3 (custom properties, animations, responsive design)
- Vanilla JavaScript (DOM manipulation, event listeners, Math.random)

## 🧠 What I Learned

- How to structure an array of objects in JavaScript
- Selecting and manipulating DOM elements with `getElementById`
- Listening for click events with `addEventListener`

- Dynamically updating text content on the page

## ⚙️ How to Run Locally

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/random-quote-generator.git
   ```
2. Open the project folder
   ```bash
   cd random-quote-generator
   ```
3. Open `index.html` in your browser — that's it, no installs needed!
